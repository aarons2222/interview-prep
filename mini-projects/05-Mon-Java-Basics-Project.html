Exercise 1: Basic Classes

Create a User class with:

  • Fields: id (int), name (String), email (String)
  • Constructor
  • Getters and setters
  • toString() method
ðŸ‘€ Show Solution 
public class User {
    private int id;
    private String name;
    private String email;
    
    public User(int id, String name, String email) {
        this.id = id;
        this.name = name;
        this.email = email;
    }
    
    public int getId() { return id; }
    public void setId(int id) { this.id = id; }
    
    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
    
    public String getEmail() { return email; }
    public void setEmail(String email) { this.email = email; }
    
    @Override
    public String toString() {
        return "User{id=" + id + ", name='" + name + "', email='" + email + "'}";
    }
}

Exercise 2: Collections & Lambdas

Create a UserManager class that:

  • Has a List<User> of users
  • Method to add a user
  • Method to find user by ID
  • Method to filter users by email domain (e.g., all "@gmail.com" users)
  • Method to sort users by name

Use Java 8 streams and lambdas!

ðŸ‘€ Show Solution 
import java.util.*;
import java.util.stream.*;

public class UserManager {
    private List<User> users = new ArrayList<>();
    
    public void addUser(User user) {
        users.add(user);
    }
    
    public User findById(int id) {
        return users.stream()
            .filter(u -> u.getId() == id)
            .findFirst()
            .orElse(null);
    }
    
    public List<User> filterByDomain(String domain) {
        return users.stream()
            .filter(u -> u.getEmail().endsWith(domain))
            .collect(Collectors.toList());
    }
    
    public List<User> sortByName() {
        return users.stream()
            .sorted(Comparator.comparing(User::getName))
            .collect(Collectors.toList());
    }
}

Exercise 3: Error Handling

Add error handling to UserManager:

  • Throw IllegalArgumentException if adding user with null name
  • Return Optional<User> from findById (instead of null)
  • Create custom UserNotFoundException
ðŸ‘€ Show Solution 
public class UserNotFoundException extends Exception {
    public UserNotFoundException(String message) {
        super(message);
    }
}

public class UserManager {
    private List<User> users = new ArrayList<>();
    
    public void addUser(User user) {
        if (user.getName() == null || user.getName().isEmpty()) {
            throw new IllegalArgumentException("User name cannot be null or empty");
        }
        users.add(user);
    }
    
    public Optional<User> findById(int id) {
        return users.stream()
            .filter(u -> u.getId() == id)
            .findFirst();
    }
    
    public User getById(int id) throws UserNotFoundException {
        return findById(id)
            .orElseThrow(() -> new UserNotFoundException("User not found: " + id));
    }
}

Exercise 4: Async Operation

Simulate an async API call:

  • Create a method that "fetches" users (add Thread.sleep to simulate delay)
  • Run it off the main thread using ExecutorService
  • Print results when complete
ðŸ‘€ Show Solution 
import java.util.concurrent.*;

public class AsyncExample {
    private ExecutorService executor = Executors.newSingleThreadExecutor();
    
    public void fetchUsersAsync() {
        executor.execute(() -> {
            try {
                System.out.println("Fetching on: " + Thread.currentThread().getName());
                Thread.sleep(2000); // Simulate network delay
                
                List<User> users = Arrays.asList(
                    new User(1, "Aaron", "aaron@test.com"),
                    new User(2, "Bob", "bob@test.com")
                );
                
                System.out.println("Fetched " + users.size() + " users");
                users.forEach(System.out::println);
                
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        });
    }
    
    public void shutdown() {
        executor.shutdown();
    }
}
